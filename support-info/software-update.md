# Internal Support Guide: Software / Firmware Update

This guide is for BeanBotics Inc. internal support staff handling software and firmware update issues for the Bean Machine.

## 1. Initial Actions
- Review the customerâ€™s description and update attempt details.
- Ask if the machine is connected to a stable network.

## 2. Information to Gather
- Customer name and email
- Product serial number
- Current firmware/software version (if provided)
- Error messages or codes
- Network type (Wi-Fi/Ethernet) and stability

## 3. Troubleshooting Steps
- Guide customer to check network connection
- Instruct on restarting the machine and retrying the update
- If stuck, perform a soft reset (provide instructions)
- Check update logs for errors
- If features are missing, verify update completeness
- If unresolved, escalate to software engineering

## 4. Documentation and Follow-Up
- Log all findings and actions in the ticketing system
- Notify engineering if issue is widespread

---
*Remind customers not to power off the machine during updates.*
