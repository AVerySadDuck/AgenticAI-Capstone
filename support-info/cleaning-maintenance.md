# Internal Support Guide: Cleaning / Maintenance

This guide is for BeanBotics Inc. internal support staff handling cleaning and maintenance issues for the Bean Machine.

## 1. Initial Actions
- Review the customerâ€™s description and cleaning history.
- Ask if non-approved cleaning agents were used.

## 2. Information to Gather
- Customer name and email
- Product serial number
- Date of last successful cleaning
- Frequency of use and cleaning
- Error messages or codes

## 3. Troubleshooting Steps
- Guide customer to empty and clean drip tray and waste bin
- Instruct on running the cleaning cycle and using approved agents
- Check for blockages in hoses or nozzles
- Verify water tank is seated and filled
- If cleaning cycle fails, check for software updates
- If unresolved, escalate to a maintenance specialist or the maintenance team

## 4. Documentation and Follow-Up
- Log all findings and actions in the ticketing system
- If escalated, notify the maintenance team

---
*Emphasize the importance of regular cleaning for machine longevity.*
