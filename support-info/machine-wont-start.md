# Internal Support Guide: Machine Won’t Start / Power Issues

This guide is for BeanBotics Inc. internal support staff handling power and startup issues for the Bean Machine.

## 1. Initial Actions
- Review the customer’s description for urgency (e.g., “machine down”).
- If urgent, escalate to the field service team or follow the critical incident protocol.

## 2. Information to Gather
- Customer name and email
- Product serial number
- Description of the issue and any error lights/messages
- Recent changes (power outages, moves, maintenance)

## 3. Troubleshooting Steps
- Confirm power cable and outlet are functional
- Check circuit breaker and emergency stop
- Ask if the machine overheated or if there was a recent power surge
- Guide customer to check fuses (if qualified)
- If unresolved, schedule a remote diagnostic or on-site visit

## 4. Documentation and Follow-Up
- Log all findings and actions in the ticketing system
- If escalated, notify the field service team

---
*Always keep the customer informed of next steps and expected timelines.*
